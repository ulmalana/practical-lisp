### Some useful key combos ###
C-h t       Tutorial
C-h i       Info/Manual
C-h *       All help related combos

C-x b       Switch buffer
C-c C-z     Back to REPL

C-x C-f     Create new file
C-x C-s     Save file
C-c C-c     Compile
C-c C-l     Compile without REPL
C-c C-k     Also compile without REPL

M-x slime   Start SLIME
, -> quit   Quit SLIME
